# Weather App Collection

This repository contains three different weather applications, each serving a different purpose and offering different features.

## Available Apps

1. **Simple Weather App** (`simple_weather.py`)
   - A basic command-line interface weather app
   - Uses wttr.in API (no API key required)
   - Auto-detects user location
   - Provides basic weather information in a simple text format

2. **Weather App with OpenWeather** (`weather_app.py`)
   - Command-line interface weather app
   - Uses OpenWeather API (requires API key)
   - Provides detailed weather metrics
   - Includes error handling and metric units

3. **Web Weather App** (`web_weather.py`)
   - Web-based weather application with a modern UI
   - Uses wttr.in API (no API key required)
   - Features a responsive design
   - Auto-detects user location
   - Displays weather information in a beautiful card layout
